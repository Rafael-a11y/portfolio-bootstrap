HTML5

O HTML5 é a quinta versão da linguagem de marcação Hypertext Markup Language, usada para estruturar e apresentar conteúdo na World Wide Web. É uma linguagem de marcação de propósito geral, que pode ser usada para criar uma ampla variedade de páginas da web, incluindo sites, aplicativos da web e jogos.

CSS3

CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para descrever a aparência de documentos escritos em HTML ou XML. Ele descreve como elementos são mostrados na tela, no papel, na fala ou em outras mídias. CSS é uma das principais linguagens da web e é padronizada em navegadores web de acordo com as especificações da W3C. Ele foi desenvolvido em níveis, com o CSS1 sendo lançado em 1996 e o CSS2 sendo lançado em 1998. O CSS3, que ainda está em desenvolvimento, está adicionando novos recursos e melhorias, como suporte para animações, responsividade e grid.

Javascript

JavaScript é uma linguagem de programação interpretada, de alto nível e dinâmica, baseada em protótipos e multiparadigma. É uma das três principais tecnologias da World Wide Web, juntamente com HTML e CSS. JavaScript é usado para adicionar interatividade e funcionalidades às páginas da web. Ele pode ser usado para criar animações, formulários, jogos, widgets e muito mais.

Bootstrap5

Bootstrap é um framework CSS e JavaScript que fornece uma base para criar sites e aplicativos da web. Ele inclui uma biblioteca de componentes de UI, como botões, menus, formulários e tabelas, bem como um sistema de grid responsivo.

Tailwind CSS

Tailwind CSS é um framework CSS de código aberto que fornece uma biblioteca de classes CSS prontas para uso. Ele é baseado no conceito de utilitários, que são classes CSS que podem ser usadas para controlar a aparência dos elementos HTML. Tailwind CSS foi lançado em 2017 e rapidamente ganhou popularidade entre desenvolvedores web. Ele é conhecido por sua curva de aprendizado rápida e sua capacidade de criar designs personalizados com facilidade.

SASS

SASS é um pré-processador de CSS que permite aos desenvolvedores web escrever código CSS mais organizado e legível. Ele fornece uma série de recursos que tornam o CSS mais poderoso e flexível.
